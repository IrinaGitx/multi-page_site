Коротко про мене

Мене звати Ірина. Я цікавлюся веб-розробкою та вивчаю структуру HTML, щоб створювати зрозумілі і доступні сторінки.

У вільний час люблю читати різні книги — від пригодницьких романів до науково-популярної літератури. Також обожнюю фотографувати природу, міські пейзажі та цікаві моменти з життя. І, звісно, для мене важливо просто проводити час з друзями, адже вони завжди додають позитивних емоцій.

Мої захоплення та навички

  • HTML та базові принципи семантики
  • Подорожування
  • ІТ

Примітка: ця сторінка створена без CSS чи JavaScript — лише чистий HTML для тренування семантики.
